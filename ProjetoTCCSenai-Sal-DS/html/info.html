Sobre o Projeto

Este projeto foi desenvolvido como Trabalho de Conclusão de Curso com o objetivo de criar um marketplace moderno, seguro e acessível.

🚀 Nossa Ideia

Criar uma plataforma inspirada em grandes marketplaces como OLX, permitindo que usuários anunciem e encontrem produtos de forma simples.

💻 Tecnologias Utilizadas

HTML, CSS, JavaScript, PHP e Banco de Dados, aplicando boas práticas de desenvolvimento web.

🎯 Objetivo do TCC

Demonstrar conhecimentos em desenvolvimento full stack, organização de sistema, experiência do usuário e estrutura de mercado online.
